# GOOGLE-Potential-Fields-
AIM

To compute the net force acting on a robot using the Artificial Potential Field method.

PROCEDURE
Initialize attractive and repulsive force values
Apply the formula: Net Force = Attractive − Repulsive
Substitute the given values
Compute the net force
Display the result
CODE
# Input values
attractive = 12
repulsive = 5

# Net force calculation
net_force = attractive - repulsive

# Output
print("Net Force:", net_force)
OUTPUT

Net Force: 7

RESULT

The net force is 7, guiding the robot toward the goal while avoiding obstacles.
